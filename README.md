# Vanad Interactions Tools

Browser-based tools for workforce management and contact center operations. Open `index.html` to browse and launch all available tools.

## Available Tools

### Call Pattern Analyzer
Analyze call center data to extract weekly and intraday distribution patterns. Upload CSV or Excel files to visualize call volumes and generate arrival patterns for workforce planning. Includes SL-Reachability correlation analysis to convert reachability targets into SL targets.

### Erlang C Calculator
Calculate staffing requirements using the Erlang C formula. Upload a call pattern from the Call Pattern Analyzer, enter volume and AHT forecasts, and compute the number of agents needed per interval to meet service level targets. Supports shrinkage adjustments, maximum occupancy constraints, and reachability-based SL target calculation.
